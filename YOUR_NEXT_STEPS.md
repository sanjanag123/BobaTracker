## Now Run These Commands (One at a Time)

### Step 1: Install Dependencies

```bash

```

- Your app will start compiling
- You'll see messages like "compiling..." 
- Eventually you'll see: `Ready on http://localhost:3000`
- **Keep this terminal window open** - your app is running!

## Troubleshooting

### If `npm install` gives an error:
- Make sure you have Node.js installed: type `node --version`
- If not, install it from [nodejs.org](https://nodejs.org/)

### If the browser shows "This site can't be reached":
- Make sure `npm run dev` is still running in the terminal
- Make sure you're going to `http://localhost:3000` (not https://)

### To stop the server:
- In the terminal, press `Ctrl + C`
